RoVidX Download Widget lets your viewers download either SD or HD MP4 files. 

== Description ==

Widget allows viewers to download the MP4 files.

Requires the [RoVidX plugin](http://wordpress.org/plugins/rovidx/ "RoVidX Plugin")

== Installation ==

1. Upload & Activate Plugin
2. Add widget to video post widget area in your theme.

== Frequently Asked Questions ==

= How does this work? =

This is an add-on widget for the RoVidX framework that allows your viewers to download SD & HD versions of your MP4 files.

== Screenshots ==

1. Adds a widget to your sidebar with both SD & HD download links.
